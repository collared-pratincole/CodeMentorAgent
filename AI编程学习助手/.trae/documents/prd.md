## 1. 产品概述

CodeMentor AI 是一款基于 VS Code 内核（Monaco Editor）构建的智能编程学习 IDE，集成国内外主流 AI 大模型，为用户提供两大核心能力：**项目逆向学习**（打开任意项目，AI 逐步教你如何从零构建）和**语言系统学习**（选择一门语言，获得全年学习计划，每日引导式学习+实战）。目标是让编程学习从"看视频"升级为"跟着 AI 动手做"。

- 目标用户：编程初学者、转行学习者、想通过阅读开源项目提升能力的开发者
- 核心价值：AI 驱动的个性化学习路径 + 沉浸式代码编辑环境，让学习与实践零距离

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 学习者 | 邮箱注册 / 游客模式 | 浏览学习内容、使用 AI 对话、打开项目分析 |
| 高级学习者 | 订阅升级 | 解锁更多模型配额、高级学习路径、项目深度分析 |

### 2.2 功能模块

1. **仪表盘页面**：今日学习任务、学习进度概览、连续打卡天数、快捷入口
2. **项目分析页面**：打开项目文件夹、AI 全景分析、逐步构建教学、交互式代码浏览
3. **学习路径页面**：语言选择、全年学习计划、每日课程、知识点+实战任务
4. **AI 对话页面**：多模型切换、代码问答、实时代码补全、上下文感知对话
5. **设置页面**：API Key 管理、模型选择与配置、学习偏好设置、主题切换

## 3. 核心流程

### 3.1 项目分析流程

用户打开项目 → 系统识别技术栈和目录结构 → AI 生成全景分析报告 → 用户选择"逐步构建"模式 → AI 将项目拆解为步骤 → 每步展示：目标说明 + 原始代码 + 讲解 → 用户在编辑器中动手实现 → AI 对比检查 → 进入下一步

### 3.2 语言学习流程

用户选择语言 → 系统生成全年学习计划 → 每日登录展示今日任务 → 知识点讲解 + 代码示例 → 实战练习（在编辑器中完成） → AI 评判与反馈 → 记录进度和成就

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深空蓝 (#0F172A) + 荧光绿 (#10B981) 作为科技感强调色
- **辅助色**：琥珀色 (#F59E0B) 用于成就/进度提示，紫色 (#8B5CF6) 用于 AI 相关元素
- **按钮风格**：圆角 (rounded-lg)，主按钮带微光效果，悬停时发光增强
- **字体**：代码区使用 JetBrains Mono，UI 使用 Noto Sans SC
- **布局风格**：左侧导航栏 + 主内容区，类 IDE 三栏布局（文件树/编辑器/面板）
- **图标风格**：Lucide 线性图标，统一 20px
- **整体风格**：暗色科技风，类似 VS Code + AI 增强的视觉体验

### 4.3 响应式设计

- 桌面优先设计，最小支持 1280px 宽度
- 编辑器区域在窄屏下可折叠侧栏
- 移动端适配：导航栏转为底部标签栏，编辑器转为只读代码展示

### 4.4 设计亮点

- AI 思考时显示流光动画效果（类似 ChatGPT 的打字效果）
- 代码编辑器与 AI 对话无缝切换，支持分屏
- 学习进度用粒子动效庆祝完成里程碑
- 暗色主题下代码区有微妙的网格背景纹理
